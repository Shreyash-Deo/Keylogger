A simple keylogger implemented in Python that logs keystrokes and saves them to a file. This project is intended for educational purposes and can be used to understand how keyloggers work, as well as to demonstrate basic keylogging techniques.


Features:


1. Keystroke Logging: Captures and logs keystrokes from the keyboard.


2. File Storage: Saves logged keystrokes to a specified file.


3. Stealth Mode: Optional feature to run in stealth mode, making the keylogger less detectable.
